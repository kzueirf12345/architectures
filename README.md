Реализация синхронизированного с тактовым сигналом модуля АЛУ на Verilog. Для запуска требуется iverilog и gtkwave.

| Команда               | Назначение    
|-----------------------|-------------------------------------|
| ```make build```      | Запускает iverilog и созданный исполняемый файл, генерирующий dump.vcd для gtkwave         |
| ```make clean```      | Очищает *.out и *.vcd          | 
| ```make start```      | Запускает gtkwave передвая ему dump.vcd        | 
| ```make all```        | build + start                          |
